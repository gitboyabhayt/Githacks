Bug Bounty Program

Help us make GITHACKS more secure. Find vulnerabilities, report them responsibly, and earn rewards for your contributions.

Submit a Bug View Scope

Program Scope

In Scope

  • Main platform (*.githacks.com)
  • Mobile applications
  • API endpoints
  • Authentication systems

Out of Scope

  • Third-party services
  • Social engineering
  • DDoS attacks
  • Physical security

Submission Guidelines

  • Include a clear description of the vulnerability and its potential impact.
  • Provide detailed steps to reproduce the vulnerability.
  • Include proof-of-concept code if applicable (no malicious code).
  • Submit only one vulnerability per report.
  • Allow up to 48 hours for initial response.
  • Do not disclose the vulnerability publicly before it is fixed.
  • Follow responsible disclosure practices.
